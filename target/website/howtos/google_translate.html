Ultra al pagina web ubi le usatores pote simplemente collar le texto pro le traduction, Google Translate provide anque un interfacie (API) que pote ser usate per le applicationes CAT, pro acceder le servicio. Pois que de decembre 2011, Google ha requirite le pagamento ab le usatores pro le uso de iste servicio. Iste Guida describe como configurar le function in OmegaT.

1. Si tu jam non ha uno, aperi un conto con google.com.

2. Seque iste ligamine pro referer te a Google pro le clave API v2 de Google Translate. Face te secur que tu ha commode le detalios de tu carta de credito. Google generara tu clave API, un linea con approximativemente 39 characteres.

3. Assigna le clave al procedura de lanceamento de OmegaT como seque, dependente si tu usa Windows, Mac, o tu lancea OmegaT ab le linea de commando o per medio de un script:

Usatores de Windows:

3.1. Locar le file de configuration OmegaT.l4J.ini (le ubication de arrangiamento initial es C:\Program Files (x86)\OmegaT).

3.2. Pro rediger iste file, tu pote haber necessitate del modo Administrator, o de mover lo temporaneemente a un differente ubication sur tu disco dur.

3.3. Aperi OmegaT.l4J.ini in un editor de texto simple

3.4. Trova le rigas sequente;

# Google Translate v2 API key
#-Dgoogle.api.key=xxxxx

E remove "#" al initio del riga -Dgoogle.api.key.

# Google Translate v2 API key
-Dgoogle.api.key=xxxxx

(ubi "xxxxx" es le clave API fornite per Google).

3.5. Salva OmegaT.l4J.ini (e si necesse, move lo al reverso al plica OmegaT).

Usatores de Mac:

3.6. Clicca con le dextero sur OmegaT.app e selige "Show Package Contents".

3.7. Trova le file OmegaT.app/Contents/MacOS/OmegaT.sh e selige tu editor preferite pro aperir le file.

3.8. Trova le rigas sequente; 

# Google Translate v2 API key
#GOOGLE_API_KEY="-Dgoogle.api.key=0123456789A0123456789B0123456789C0123456789D"
E remove "#" al initio del riga. Ce riga defini GOOGLE_API_KEY.

# Google Traducer v2 API clave
GOOGLE_API_KEY="-Dgoogle.api.key=XXXXXXXX"

ubi "XXXXXXXX" es tu clave API de Google Translate.

3.9 Salva OmegaT.sh.

Usatores de Linux:

3.10 Face te certe que tu ha le lanceator Kaptain installate (refere te si necessari al Guida de Linux), deinde scribe le clave API de Google Translate in le pertinente campo sur le scheda Connexiones.

Usatores que lancea OmegaT per le linea de commando o per un script:

3.11 Adde le argumento sequente al commando de lanceamento:

-Dgoogle.api.key=XXXXXXXX

ubi "XXXXXXXX" es tu clave API de Google Translate.

Le commando basic de lanceamento con le API de Google Translate in consequentia essera:

java -jar -Dgoogle.api.key=XXXXXXXX OmegaT.jar

4. Configuration in OmegaT

4.1. Lancea OmegaT

4.2. Selige Optiones > Traduction Automatic

4.3. Marca Google Translate v2 (lassante non marcate le altere optiones)

5. Nota:

5.1 Le traductor automatic Google Machine Translation es un servicio pagate (US$20.00 per million de characteres). Tu pote controlar le uso quotidian ab tu conto Google. In le menu precedente:

More > Even More > Innovation/Code > APIs Console > Service/Translate API
