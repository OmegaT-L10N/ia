Iste Guida forni informationes re le uso de OmegaT sur Linux Illo es in particular mirate al usatores qui es nove de Linux.

Java

OmegaT non flue ben sur tote versiones de Java. Si Java es jam installate sur tu systema que es probabilemente le caso) e tu projecta de usar iste existente installation Java de systema in conjunction con OmegaT, in prime loco controla que illo es convenibile. Pro facer isto, aperi un terminal (alsi note como un consola o fenestra pro linea de commando; programmas commun pro isto es konsole (KDE) e gnome-terminal (GNOME)), e scribe:

java -version

Le systema emittera detalios del version de Java jam installate a nivello de systema.

Versiones Java Oracle (un vice Sun): controla le requisitos del systema in le documentation de OmegaT (readme.txt) pro le precise version de Oracle Java requirite (i.e. si tu version de Java es satis recente).

Versiones Java de tertie parte: usque le tempore relativemente recente, OmegaT functionava sol sur Java proprie de Oracle/Sun. OmegaT ora fluera sur le installation Java OpenJDK supplite con alicun distributiones de Linux, ma alicun functiones (assi como le suite de optiones del linea de commando) pote non esser disponibile; pote esser supplite un serie de typo de characteres; e qualque defectos esseva reportate, assi como con le function de monstra de fenestra e de segmentation.

Si tu es insecur si tu version de Java es convenibile, tu pote:

  • Simplemente installar un version de OmegaT sin Java e vide si illo es ben functionante. Si illo non functiona ben, tu ancora ha le option de installar un version local de Java o un differente version de Java a nivello de systema.
  • Installar un secunde, version de Java non ordinari, a nivello de systema, e lancear OmegaT explicitemente usante iste version.
  • Installar le version de OmegaT con Java includite.

Si tu constata que le version de Java installate a nivello de systema non es compatibile con OmegaT e tu desidera reimplaciar iste version de Java con le version Oracle de Java, controla in prime loco si es disponibile un pacchetto Oracle pro tu distribution de Linux (SUSE, Ubuntu, Mint etc.). Si assi, isto rende le installation multo plus facile.

Si tu obtene le version Oracle de Java directemente ex Oracle, le installation essera probabilemente correcte, ma tu debera lo configurar manualmente pro lo render le version ordinari, a nivello de systema. Isto non es recommendate, in particular si tu non es familiar con Linux.

Le tertie option, id es installar un version de OmegaT con Java includite a nivello local, es semper le option "secur", e es recommendate.

Selectionar e discargar OmegaT

Le versiones sequente de OmegaT es disponibile pro Linux:

"Standard", "Latest", "/trunk" e "Webstart". Le version "Latest" es recommendate. (refere al pagina "Discargas" sur le sito web de OmegaT pro detalios del differentias.)

Sin/con JRE includite, id es le multe platteforma sin JRE, e.g. "OmegaT_2.1.7_02_Beta_Without_JRE.zip" e le version specific pro Linux con JRE, e.g. "OmegaT_2.1.7_02_Beta_Linux.tar.bz2". Selige un version sin o con JRE dependente sur tu situation Java (vide supra).

Como application open-source, OmegaT esseva includite per plure distributiones de Linux in lor repositorios, assi tu pote esser capace a lo installar per iste via, ma a vices il non ha un version actualisate. Nota totevia que:

  • le major parte del distributiones Linux usa le version de Java OpenJDK. Al momento del script, OmegaT currera multo melio con le version de Java Oracle (vide supra).
  • Le version OmegaT habitualmente supplite es le version "Standard". Plus de functiones habitualmente essera trovate in le version "Latest".

Nota que il es facile installar versiones multiple de OmegaT sur Linux sin conflictos. Illos totevia accedera semper al mesme files de configuration del usator (a minus que lanceate ab un conto del usator differente).

Installar OmegaT

Le version Linux de OmegaT ora es supplite con su proprie script de installation: linux-install.sh. Iste script:

  • loca le files programma de OmegaT in /opt/omegat/OmegaT-<version>;
  • include le scripts de lanceamento bash e kaptain;
  • crea le ligamines ab /usr/local/bin al scripts de lanceamento.

Le script bash es un file multo basic que pote esser modificate manualmente con optiones de linea de commando pro lancear e configurar OmegaT.

Le script Kaptain (o "grammatica", como le scriptes kaptain es denominate) te habilita a seliger le optiones del lanceamento e del configuration per un facile interfacie graphic. Tu pote per exemplo lo usar pro seliger le lingua del interfacie de usator, pro assignar plus de memoria a OmegaT, o pro inserer le contrasigno de Google Translate. Tu pote personalisar le script lanceamento de kaptain de maniera que tu preferentias sia le configurationes initial; redige simplemente le file omegat.kaptn in un editor de texto. Le file es commentate pro te adjutar. Pro facer functionar le script kaptain, comocunque, tu debe primo installar kaptain. If illo non es in le repositorio de tu distribution, tu pote lo discargar ex kaptain.sourceforge.net.

Instantaneo del schermo del lanceator kaptain de OmegaT

Le script de installation provide convenientia additional, ma tu pote ancora installar OmegaT sur Linux manualmente si tu lo prefere. Simplemente dispacchetta le archivo in un plica de tu selection. Le scripts de lanceamento non essera scripte in tu percurso de execution in iste caso.

Lancear OmegaT ab le linea de commando

OmegaT non debe esser lanceate per linea de commando, ma facer assi pote esser utile pois que illo forni informationes pro le cernita del defecto. Pro lancear per linea de commando, aperi un fenestra terminal (non es necesse permissiones de radice). Naviga al plica continente le file OmegaT.jar, e.g.:

cd /opt/omegat/OmegaT_2.3.4

e confirma con ENTER.

Ergo inscribe le commando ipse de lanceamento, e confirma con ENTER. Le commando basic de lanceamento es:

java -jar OmegaT.jar

Le commando es sensibile al majusculas.

Si tu fide, o desira usar, un installation Java non installate a nivello de systema, isto non essera sufficiente: tu debe anque instruer tu systema a ubi trovar Java. Si tu ha installate un version de OmegaT con supplite JRE, le script lanceamento supplite jam contine iste percurso pro te. Si tu ha installate un altere version de Java, ma tu non ha rendite lo como version ordinari a nivello de systema, tu debe includer le integre percurso del file binari Java in le commando de lanceamento. Isto essera simile a /usr/lib/jvm/java-6-sun-1.6.0.16/jre/bin/java; prende nota de ubi Java es installate durante le installation, a fin que tu pote lo retrovar. Le commando essera dunque (in iste caso):

/usr/lib/jvm/java-6-sun-1.6.0.16/jre/bin/java -jar OmegaT.jar

In tu script de lanceamento, non confunder le percurso a OmegaT.jar e le percurso al commando Java. Le primo (con le commando cd) es necesse si tu script de lanceamento non es in le mesme plica que OmegaT.jar. Le secundo es necesse si le version Java que tu desidera usar non es le installation base a nivello de systema.

Lancear OmegaT per un script de lanceamento

Como jam mentionate, un script de lanceamento bash es fornite con OmegaT. Un vice que isto esseva copiate o ligate a tu percurso de execution (facite automaticamente si tu usa le script de installation fornite), tu pote lancear OmegaT per le Alt+F2, pois scribente omegat e confirmante con ENTER.

De norma le script de lanceamento es executabile, ma tu systema pote mutar su permissiones (e.g. durante le disincassamento del archivo) e tu poterea deber lo render (executabile) de novo.

Lancear OmegaT facilemente

Pro render lanceamento OmegaT mesmo plus facile, adde lo a tu menu de lanceamento, o pone un icone de lanceamento pro illo sur tu scriptorio o in tu quadro scriptorio. Le possibilitates es numerose, e le proceduras pro los implementar non minus numerose, debite al plethora de distributiones.

Apparentia del typos de character in OmegaT sur Linux

Le apparentia del typos de character in OmegaT sur Linux pote differer largemente, secundo:

  • Le version Java usate (le identic typo de character pote parer assatis differente sur le versiones Sun e OpenJDK de Java)
  • Le typos de character disponibile/seligite
  • Le mesura del character
  • Si es o non habilitate le anti-aliasing (isto es un option de lanceamento a linea de commando, ma tu pote alsi adjustar lo in le script de lanceamento kaptain)

Disinstallar OmegaT

Pro remover omne tracias de OmegaT ex tu systema:

Dele le plica del programma OmegaT. Isto potera facilemente esser restaurate novemente in le futuro.

Dele le plica /home/<user>/.omegat de omne conto de usator ab le qual OmegaT esseva lanceate. Isto delera le definitiones de configuration de omne usator que esseva facite (como le cambios al regulas de segmentation), e â€“ si le plica es delite "debitemente", id es. non simplemente movite al antiqualia, non potera ser restaurate.

Dele omne scripts de lanceamento e leva omne entratas personalisate pro OmegaT in tu scriptorio.

Disinstallar OmegaT non removera alicun projectos OmegaT ab tu systema, a minus que tu poneva tal projectos intra le plica(s) del programma OmegaT o del files de configuration, le qual non es un bon ubication pro illos in omne caso.
