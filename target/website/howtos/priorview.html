Scenario de fundo e principios

Priorview es un parve utilitate que provide un function vista preliminar pro uso con OmegaT. Pro le scenario de fundo e le principios, refere al Guida al vista preliminar. Iste Guida describe le uso de Priorview pro Windows e Linux. Si tu es usante Mac OSX, tu pote discarga un version de Priorview pro Mac hic. Le version pro Mac OSX non es ancora documentate e le information in sequito non referer se a illo.

Software necessari

Pro usar Priorview sur Windows o Linux, tu necessita le sequente:

1. Un lector del tela, specificamente o:

  • Opera
    or
  • Firefox conjunctemente con le add-on pro Firefox "ReloadEvery"
    Installa in prime loco Firefox (si tu non ha lo jam installate), pois installa ReloadEvery.
  • (Discargar ex le ligamines fornite; le usatores de Linux pote preferer de installar lo ex lor repositorios de distribution, ubi isto es possibile.)

2. Un utilitate de conversion, specificamente o:

  • Abiword
    or
  • LibreOffice conjunctemente con le plugin Unoconv

3. Tcl/Tk

Quasi omne version actual de Tcl/Tk essera probabilemente commode, ma tu debe haber tote le duo Tcl (le interprete del script) e Tk (le componente graphic).

4. Priorview

Pro installar Priorview, discassa simplemente le archivo zip in un location convenibile sur tu disco dur. Pro lancear Priorview, aperi un prompte/terminal/consola de commando e inscribe:

cd path

ubi percurso es le nomine percurso complete del plica in le qual tu poneva preview.tcl. Deinde scribe:

wish priorview.tcl

Tu pote simplificar le lanceamento con le script de un file script, id es un file de texto simple continente iste duo lineas.

Usar Priorview

Si toto vade ben, Priorview deberea apparer quando lanceate como ante describite. Sur isto, selige:

  • le nomine del final file (in le plica /target in tu projecto OmegaT);
  • le lector del tela que tu ha installate;
  • le frequentia con le qual le script controla si iste file cambiava (intervallo de polling);
  • le utilitate de conversion file que tu ha installate.

Quando tu ha seligite tote istos, compare le button Start. Quando actuate, isto lancea tu lector del tela con un pagina simulacro instruente te a preparar le function de renovation automatic, quem tu deberea dunque facer, e pois pulsar Ctrl+s, Ctrl+d in OmegaT pro lancear le renovation automatic del lector del tela.

Durante que Priorview es functionante, tu pote:

  • interrumper e reinitiar lo;
  • seliger un file nove (Si tu es laborante in un projecto continente plure files, tu non debe relancear Priorview quando tu attinge le fin de un file; tu pote seliger simplemente le sequente);
  • cambiar le intervallo de polling.

Un vice que Priorview esseva lanceate, comocunque, tu non pote seliger un lector del tela differente o un utilitate de conversion; iste optiones in consequentia es occulte post le lanceamento. Pro seliger un lector del tela differente o utilitate conversion, tu debe clauder Priorview e relancear lo.

Le button Quita claude Priorview e dele alsi le files temporari que illo creava. (Altere methodos de clauder Priorview non delera le files temporari.)

Un edition de pressa del Commission Europee (in formato .odt) in OmegaT e in sequito a illo le instantanee texto final monstrate in Opera. In basso a dextra: le utilitate Priorview. Le arrangiamento monstrate hic es justo pro gratia de exemplo; tu pote naturalmente arrangiar le fenestras como tu vole.

Chassa al defecto

Si le percurso allocation de priorview.tcl contine spatios, pone lo inter virgulettas, exempli gratia:

cd "C:\My Files\priorview"

Si Priorview falle al lanceamento:

Dependente sur le version de Tcl/Tk que tu ha installate, tu pote deber adder le numero de version al commando desirate, exempli gratia.:

wish8.5 priorview.tcl

Paga reguardo al characteres minuscule/majuscule.

If the file monstrate in le lector del tela non esseva renovate, controla le sequente:

  • Que tu recordava de pulsar Ctrl+s, Ctrl+d in OmegaT. usque/sinon tu face lo, il essera nulle cambios a discoperir/monstrar!
  • que le utilitate de conversion de file es installate debitemente e es functionante. Tu pote probar isto con le commando pertinente in sequito sur le linea de commando (terminal/consola/prompte DOS). In cata caso, file es le nomine de un file .docx o .odt. Si tote es ben, un version .html del file essera producite:
    Pro Abiword:
    abiword --to=html file
    For LibreOffice+Unoconv:
    unoconv -f html file
  • Que tu ha preparate le intervallo de auto refrescamento in le lector del tela.
